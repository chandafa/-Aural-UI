# Changelog

All notable changes to Auralix UI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-11

### Added

- Enhanced Modal with Framer Motion spring animations
- Enhanced Accordion with smooth height animation
- Enhanced Dropdown with AnimatePresence transitions
- New "Developer Experience" code showcase section on landing page
- Futuristic hero section with perspective grid background
- JSON-LD structured data for SEO
- Google Search Console verification support

### Changed

- Unified sitemap and robots.txt to use consistent domain
- Optimized AnimeDemo with IntersectionObserver for better scroll performance
- Updated anime.js integration to v4 API

### Fixed

- Fixed "anime is not a function" error with anime.js v4
- Fixed TypeScript lint errors in Modal component

## [1.0.0] - 2026-01-06

### Added

- Initial release with 87+ components
- Full TypeScript support
- Dark mode support
- Tailwind CSS v4 compatibility
- GSAP, Framer Motion, React Spring, and Anime.js animations
- Comprehensive documentation site
